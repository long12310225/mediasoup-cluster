### 客户端启动流程
 ```
 根据客户端启动指令
 npm run dev
 结合TypeScript13的改动得知，启动文件是src/app/page.tsx
 ```

 具体流程
 ```mermaid
 sequenceDiagram
    participant Client as 客户端
    participant Master as 主信令服务
    participant Producer as 从生产服务
    participant Consumer as 从消费服务
    Client->Client: 创建CreateRoom、JoinRoom、DeleteRoom组件，并绑定onSuccess事件
 ```
